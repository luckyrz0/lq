# 二维数组改造代码
1. - 定义一个二位数组，大小分别表示四个方向，每个方向有三张图片
```java
   Image img[][]=new Image [4][3];
```
2. - 根据二位数组的特性，把图片进行重命名
3. - 给二维数组赋值
```java
for(int i=0;i<img.length;i++){
	for(int j=0;j<img[i].length;j++){
			img[i][j]=Image.createImage("/sayo"+i+j+".png");          
			}
```
4. - 把图片的赋值用二维数组来代替
